############ INSTALL GANTTPLANOPT Runner ################

1. Add compiled programm GanttPlanOptRunner.* - Files to root folder of GanttPlanOptRunner
2. Add folders "Continuous" and "Init" to GanttPlan root folder and add all *.config and *.lang files to both folders
3. Make sure the reference of local.config file in both folders are correctly referenced to the config file from Config-Folder 
        - C:\Program Files\GANTTPLAN_V6\Config\GanttPlanContinuous.config in Continuous
        - C:\Program Files\GANTTPLAN_V6\Config\GanttPlanInit.config in Init
